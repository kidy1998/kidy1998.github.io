프로젝트 목표

상영 중인 영화 작품 뿐만 아니라 다양한 OTT서비스에서 제공하는 영화 정보를 설명하며, 서울 기준으로 3대 영화관의 위치 및 시설정보를 제공하고 유저들이 이를 직접 평가하는 영화-영화관 통합 사이트를 제작하고자 한다.

프로젝트 소요 기간: 약 4주

구성 인원: 5인

사용 기술

Backend : Java, JSP

Frontend : CSS, JavaScript

Database : Oracle

Environment : Spring, Mybatis, Apache Tomcat

Collaboration : Github,Figma

담당 부분: 영화관, 관리자 페이지에 대한 UI, 백앤드 부분 구현
