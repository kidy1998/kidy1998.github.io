프로젝트 목표

카페에서 홍보하고 싶은 메뉴와 홍보 시간, 할인율 등을 입력하면 chat gpt 가 조건에 따른 홍보 멘트를 생성하여 정해진 시간마다 홍보 멘트를 발화하고 손님이 왔을 시 인원수와 현재 시간에 따라 자리를 안내하는 로봇을 구현하고자 한다. 이때 홍보 메뉴를 등록하고 시나리오에 따라 저장된 내용을 발화하는 관리자 시스템을 제작한다.

프로젝트 소요 기간: 4주

구성 인원: 2인

사용 기술

Backend : Java, JSP

Frontend : CSS, JavaScript

Database : Mysql

Environment : Spring Boot, GCP

Collaboration : GitLab, Figma

API : Chat GPT

담당 부분: 관리자 페이지의 백엔드 부분 구현, GCP 를 통한 배포
